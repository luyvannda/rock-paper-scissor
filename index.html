Rock Paper Scissor

Pseudocode

  1. Start the game
  2. Ask the user to choose rock, paper or scissors
  3. Generate a random number between 0 and 2 to represent the CPU's choice
  4. Compare the user's choice with the CPU's choice
  5. If the choices are the same, it's a tie
  6. If the user chooses rock and CPU chooses scissors or if the user chooses paper and CPU chooses rock or if the user chooses scissors and CPU chooses paper, then the user wins Otherwise, CPU wins
  7. Ask if the user wants to play again
  8. If yes, go back to step 2
  9. If no, end the game
